# Mesto

## Описание проекта

Mesto — это интерактивное веб-приложение, представляющее собой социальную сеть для обмена фотографиями. Пользователи могут добавлять новые места с изображениями, редактировать свой профиль, ставить лайки карточкам других пользователей и удалять свои публикации. Проект разработан с использованием современных веб-технологий и демонстрирует навыки фронтенд-разработки.
https://liziwinc.github.io/mesto-project-ff/

## Особенности проекта

- **Адаптивный дизайн**: Приложение корректно отображается на различных устройствах, от мобильных телефонов до десктопов.
- **Интерактивные модальные окна**: Попапы для редактирования профиля, добавления новых мест и просмотра изображений с плавными анимациями открытия и закрытия.
- **Валидация форм**: Реализована клиентская валидация форм с отображением ошибок в реальном времени и управлением состоянием кнопок отправки.
- **Взаимодействие с сервером**: Полная интеграция с API для загрузки данных пользователя, карточек, обновления профиля, добавления/удаления мест и управления лайками.
- **Модульная архитектура**: Код организован по модулям (API, карточки, модальные окна, валидация), что обеспечивает читаемость и поддерживаемость.
- **BEM-методология**: CSS написан с использованием BEM (Block Element Modifier) для структурированного и масштабируемого стилизации.

## Как запустить проект

### Предварительные требования

- Установленный Node.js
- npm

### Установка и запуск

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/your-username/mesto-project-ff.git
   ```

2. Перейдите в директорию проекта:
   ```
   cd mesto-project-ff
   ```

3. Установите зависимости:
   ```
   npm install
   ```

4. Запустите проект в режиме разработки:
   ```
   npm run dev
   ```
   Приложение будет доступно по адресу `http://localhost:8080`.

5. Для сборки проекта в продакшн:
   ```
   npm run build
   ```
   Собраные файлы будут находиться в папке `dist`.

6. Для деплоя на GitHub Pages:
   ```
   npm run deploy
   ```

## Структура проекта

- `src/` — исходные файлы
  - `index.html` — главная HTML-страница
  - `index.js` — основной JavaScript-файл
  - `blocks/` — CSS-компоненты по BEM
  - `images/` — изображения
  - `pages/` — стили страниц
  - `scripts/` — JavaScript-модули (API, карточки, модальные окна, валидация)
  - `vendor/` — сторонние стили и шрифты
- `webpack.config.js` — конфигурация Webpack
- `babel.config.js` — конфигурация Babel
- `postcss.config.js` — конфигурация PostCSS
